# Voidscape Linux Getting Started

The supported local workflow is documented in `docs/DEVELOPMENT.md`. Use the wrapper scripts from the repository root:

```bash
scripts/build.sh
scripts/run-server.sh
scripts/run-client.sh
```

Create launch accounts through the Voidscape account portal when testing public launch flows. For local development, follow the database and preset notes in `docs/DEVELOPMENT.md` and `docs/CONFIG-MATRIX.md`.

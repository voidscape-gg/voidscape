# Voidscape Public Source

This repository is the public AGPL source mirror for Voidscape, a RuneScape Classic private server forked from OpenRSC Core-Framework.

The live website Source tab points here so players can inspect the code baseline, served download hashes, and launcher manifest proof without relying on a personal GitHub account.

Generated runtime files, logs, local databases, backups, private deployment metadata, and developer memory files are intentionally excluded from this mirror.

# Voidscape Discord Assets

Standalone Discord community assets. These are not wired into the game client or server.

Files:
- `server-icon-512.png` - Discord server icon and social avatar.
- `server-banner-1920x1080.png` - high-resolution server banner source.
- `server-banner-960x540.png` - Discord's recommended static server banner size.
- `invite-splash-1920x1080.png` - invite splash/background source.

Discord upload notes:
- Server icon: Server Settings > Overview > Server Icon.
- Server banner: Server Settings > Overview > Server Banner Background.
- Invite splash: Server Settings > Overview > Invite Splash Background, if available for the server.

The banner and splash intentionally avoid logo text and keep the top area subdued so Discord's server title remains readable.

# Voidscape Art Brief

Use this whenever ChatGPT is acting as the concept studio for Voidscape.

## Style

- RuneScape Classic scale and readability.
- Mostly authentic RSC silhouettes with custom void corruption.
- Dark stone, black robes, bone, obsidian, violet, magenta, cyan highlights.
- Chunky filled pixel-art forms for sprites and icons.
- Strong outline and readable silhouette at tiny in-game size.

## Avoid

- Modern OSRS/RS3 proportions unless explicitly requested.
- Smooth digital-paint gradients on sprites.
- Thin line-art icons with empty interiors.
- Tiny unreadable text baked into generated art.
- UI buttons, labels, menus, or system icons inside generated art unless the
  whole image is meant to be a background plate.

## Asset Extraction

- For cutouts, request separate images on pure `#00FF00` green-screen.
- For inventory icons, request transparent PNGs or green-screen cutouts with no
  baked drop shadow.
- For NPCs and bosses, ask for a concept first, then a strict sprite-sheet pass.
- Re-upload the selected concept before requesting individual assets from it.

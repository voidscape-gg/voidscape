Voidscape whitepaper

A classic game loop, finished with intent.

You have seen servers open, spike, and shut down. Some teams loved RuneScape Classic and lacked the engineering. Some teams marketed well and lacked design. Voidscape keeps the old rhythm, then connects the loops Jagex left half-built.

Economy loop

Skillers feed combat. Combat feeds skillers.

Skillers make supplies. Combat players burn them. Void PvM, Bounty, and risky play return raw goods. The Auction House prices the next grind, and the rewards flow back into the loop.

The thesis

RSC was magic. It was also unfinished.

We keep the movement, tradeoffs, equipment identity, Wilderness pressure, staking culture, resource grind, and strange social rhythm. Then we add pressure to the quiet parts so supplies, risk, rewards, and markets push players toward each other.

Authentic first Core systems stay true to RuneScape Classic.
Loops over gimmicks Every addition strengthens the player-driven loop.
Visible risk Wilderness, Bounty, and Void Dungeon make reward feel earned.
Low-friction start Starter paths get new players into the game fast.

Progression

Fast combat, meaningful skilling.

Combat trains at 10x, skilling trains at 2x, and a subscription card adds +1x to both. PK-ready accounts appear fast; high-volume skilling goods keep value because the grind still matters.

Custom systems

Risk, practice, and reward all need visible objects.

Players should see the loop working in the world: catch targets, train timing, chase beams, carry keys, and open chests inside dangerous places.

Void branch

Custom power belongs in Void PvM.

Void items feel rare and leave rune, arrows, food, and familiar tactics intact. We use them as anti-Void tools: you fight Void monsters, chase Void Keys, and improve risky PvM. Against players, visible stats rule.

Principle No hidden global power creep. Void effects are for Void monsters; PvP and staking stay familiar.
Void Scimitar Anti-Void melee chase weapon.
Void Shortbow No-arrow behavior only against Void NPCs.
Void Amulet Boosts stackable drops from Void NPCs only.
Void Mace Crush-style Void PvM niche.

Skill hooks

Small changes, real reasons to skill.

We reuse RSC actions players already understand, then give those actions more buyers.

Firemaking -> Prayer Oak through magic logs leave tiered ashes. Offer them at altars for scaling Prayer XP.
Woodcutting -> Sparrow 90+ Woodcutting can find a tradable Void Sparrow from yew and magic logs.
Fletching -> Ammo Better logs make more shafts, and Void rewards return arrowheads for fletchers to finish.
Mining -> Smithing Void loot leans into ores, coal, gems, and processable goods. Smiths finish the value.
Smithing -> Arrowheads x50 arrowhead batches make smiths a real part of the ranged supply chain.
Prayer -> Combat demand Bones and ashes stay valuable because serious fights need Prayer levels and recharge access.

Client access

Custom first. Complexity earns its place.

The launcher updates the desktop client when testers press Play. The Android app shares the same game core so mobile players get the same rules. The HUD, bank, inventory, chat, settings, world map, and onboarding have been rebuilt around the Voidscape identity while keeping the old game readable.

Auto-updating launcherBeta testers get new client/cache builds through the manifest when pressing Play.
Android clientNative Android shell, shared RSC core, emulator smoke coverage, and portal handoff for account creation.
Workbench QACommand and screenshot tooling helps us test features through repeatable scenarios.

Roadmap discipline

Ship the loop, then expand the world.

Voidscape already has a lot: starter paths, Void Council intro, subscription cards, Auction House, Bounty Hunter, Void Dungeon, Void Keys, rare beams, titles, scouting, Android, and a custom UI. Next we pressure-test the loop and add features that strengthen it.

Beta tuningUse playtests, AH prices, and balance reports to adjust drops and supply pressure.
Content toolsMake custom NPC, item, sprite, and map work easier so quality rises and updates keep moving.
Boss standardPause weak boss attempts until art, animation, arena, and mechanics can meet the Voidscape bar.
Public launchKeep registration low-friction while protecting free starter cards before they enter the trade economy.

Build proof

The features are in the client, not just the pitch.

Onboarding, path choice, HD visual controls, Void story staging, XP-on-hit, beams, keys, chests, and catching practice are already visible in beta screenshots.
